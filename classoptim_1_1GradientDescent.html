Detailed Description

template<typename fp_t, bool use_prox = false>
class optim::GradientDescent< fp_t, use_prox >

Gradient Descent.

Gradient Descent is a first-order optimization algorithm that using negative gradient as the line search direction. You can specify the line search criterion, step scheduler and accelerator.

Template Parameters
fp_tfloating-point type
use_proxwhether to use proximal operator
